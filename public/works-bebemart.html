BeBe-Mart(架空)

Design/Coding

育児中の親をターゲットにした、育児道具専門の売買マーケットサイトです。
このサイトでは対象の年齢を過ぎた育児道具を買い取ってもらったり、他の人が出品した商品を購入することができます。

課題
「育児道具の使用期間に対しての出費が大きい」
「使い終えた育児道具が場所をとっていてなんとかしたい」という問題を解決する
ターゲット
0歳から3歳までの育児中の親
制作期間
4ヶ月
使用ツール
Figma / illutrator
使用言語
HTML5 / CSS / JavaScript
作品はコチラのリンクから
